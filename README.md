# Online-Quiz-Platform
An Online Quiz Platform


An online platform to take quizzes. Built using Nodejs.

## Running server
Clone the repo:<br>
`git clone https://github.com/SagarB-97/Online-Quiz-Platform.git`
<br><br>
Install dependencies: <br>
`npm install`
<br><br>
Run Server using : <br>
`npm start`
